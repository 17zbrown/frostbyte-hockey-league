Game Reports

Both clubs must report the same score
Enter the full box score for both clubs — goals, assists, shots, hits, PIM and goalie saves — straight off your EA end-of-game screen. It publishes only when it mirrors the opponent GM's entry. If they don't match, both GMs get 24 hours to align their box scores; if the window closes still unmatched, the game is a forfeit for both clubs. (You can also send your EA screenshots to the commissioners.)
